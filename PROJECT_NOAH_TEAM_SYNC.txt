============================================================
PROJECT VISION
============================================================

Project Noah is a planetary-scale animal intelligence network. Powered by orbital thought-translation satellites, it gathers, translates, and maps animal thoughts worldwide, providing real-time insights into behavior, emotion, migration, and ecosystem health.

The app should feel like Google Earth meets Doppler radar meets a wildlife intelligence dashboard.

The user should land directly on a global satellite map, not a card grid. The satellite map is the main product experience.

Users can explore Earth from above, zoom into regions, see animals in their real environments, filter animal activity, and read thought bubbles when they get close enough.

============================================================
CORE EXPERIENCE
============================================================

1. User opens Project Noah.
2. They see a dark, satellite-style global map.
3. Animal activity markers appear across the world.
4. The left sidebar lets users filter by animal, mood, emotion, region, and search terms.
5. The bottom dashboard shows live stats based on the visible map area.
6. As the user zooms closer, animals become more detailed.
7. At close zoom levels, thought bubbles appear above animal markers.
8. Clicking an animal opens more detail about that animal, including thought history.

The priority is to make the map feel like a live satellite intelligence system.
